Hostel Finder
Introduction
Welcome to Hostel Finder! This project aims to help users find hostels in their desired location. It is developed using JavaScript.

Installation
To get started with this project, follow these steps:

Clone the repository:
git clone https://github.com/Bingum-de-Alwis/Hostel-finder-.git
Navigate to the project directory:
cd Hostel-finder-
Usage
Once you have cloned the repository, you can start using the application:

Open the index.html file in your browser to view the application.
Follow the on-screen instructions to search for hostels in your desired location.
